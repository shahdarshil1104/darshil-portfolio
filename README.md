# Darshil Shah — React Portfolio (Vite)

## Run locally
```bash
npm install
npm run dev
```

## Customize content (no component edits needed)
Edit these JSON files:
- `src/data/profile.json`
- `src/data/projects.json`
- `src/data/experience.json`
- `src/data/skills.json`
- `src/data/education.json`

## Resume
Your resume is included at `public/resume.pdf`.

## Deploy
- Vercel / Netlify: build command `npm run build`, output `dist/`.
